# How-To Guides

This section is task-oriented documentation for people who want to add or customize map functionality without first reading all of the architecture docs.

If you are new to the codebase, start here.

Recommended reading order:

1. [Add a Layer](./add-a-layer.md)
2. [Add a Popup](./add-a-popup.md)
3. [Add a Layer Family](./add-a-layer-family.md)
4. [Add a Project](./add-a-project.md)

These guides are intentionally practical:

- what files to touch
- what order to do things in
- what patterns already exist in the app
- what to test afterward

For the underlying concepts and system diagrams, go back to:

- [Architecture](../architecture.md)
- [Layers](../layers.md)
- [Popups](../popups.md)
- [Projects](../projects.md)
- [Layer Families and Raster Layers](../raster.md)
